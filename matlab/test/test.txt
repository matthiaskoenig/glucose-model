********** MODEL NAME
Borisuk98 Frog egg model

********** MODEL NOTES
Published in: Borisuk MT, Tyson JJ, 1998, Bifurcation analysis of a model of mitotic control in frog eggs, J Theor Biol. 1998 Nov 7;195(1):69-85.                                                                     
Model type: ODEs and Algebraic Equations based on mass action and Michaelis-Menten kinetics                                                                                                                           
                                                                                                                                                                                                                      
SBtoolbox model created by Henning Schmidt and modified 2007-02-03 by T. Nordling, tn@kth.se
